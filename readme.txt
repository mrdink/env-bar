Display a colored bar to help distinguish between different environments.

== Description ==

Env Bar is a great way to keep track of your server environment. On the Env Bar Settings page you can select Development, Staging, or Production to then output a colored bar in the bottom of your site.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/env-bar` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Environment screen to configure the plugin

== Frequently Asked Questions ==

= Can you use this plugin with Multisite? =

Yes, this will add the bar globally and you can access the options page within the Network.

= Can you use this plugin as a mu-plugin? =

Yes, just upload the env-bar.php to your /wp-content/mu-plugins/ directory.
